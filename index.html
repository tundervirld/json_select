Json to Select

En el desarrollo de aplicaciones web que están orientadas a la integracion y dinamismo, es necesario el uso de API'S para ofrecer una experiencia de ususario mas elaborada. Por esta razon jsonSelect es una herramienta de facil integracion en tu proyecto web que permitira inyertar a manera de JSON los valores que necesites en un select y la vez nombrar como necesites las propiedades a manera de atributo en el caso de necesitar que la misma seleccion te devuelva varias alternativas que le hagan referencia. Para que quede mas claro el ejemplo hemos planteado un ejercicio que desde un Json que contiene los paises de Sub America y algunas de las caracteristicas de estos, sera pasado a nuestro plugin JsonSelect, nombraremos los atributos del json a nombres que nos faciliten la comprension y estraccion de informacion con respecto al pais seleccionado en el select.

Por ejemplo si no desea que un campo que viene en el json se genere como un atributo en el json select, debera colocarlo dentro de la configuracion de los atributos y setearlos como oculto ya que el valor por defecto es nombrarlo con el mismo nombre que viene del Json. Si no se especifica ningun valor en la configuracion de los atributos, se consideran como el atributo value el que venga dentro de los atributos del JSON con el nombre de llave "ID", si no viene ningun campo con el nombre ID, jSonSelect generará un id incremental, como recomendacion es pararle un atributo con llave "ID", para poder tratar los resultados con mayor presicion y control por el desarrollador que incorporo JsonSelect a su proyecto.
